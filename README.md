# 🛒 Amazon Clone

A fully responsive **Amazon Clone** built using modern web technologies. This project replicates the core user interface and basic functionality of Amazon’s e-commerce platform with a focus on clean design and performance.

## 📌 Features

- 🏠 Home page with product listings  
- 🔍 Product search functionality  
- 🛍️ Add to cart and remove from cart  
- 🧾 Product details section  
- 📱 Fully responsive (mobile, tablet, desktop)  
- ⚡ Fast and optimized UI  
- 🎨 Amazon-inspired modern design
